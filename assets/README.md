# Assets Directory

This directory contains app icons and splash screens.

## Required Assets

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Generating Assets

You can generate these assets using:
1. Figma or Adobe Illustrator
2. Online tools like https://appicon.co
3. Expo's asset generation tools

For now, placeholder assets will work for development.
