## ðŸ“Š Key Features (Marketing Copy)

### Core Features
1. **AI-Powered Receipt Scanning**
   - Camera + photo library support
   - OCR extraction (amount, date, merchant)
   - Auto-categorization

2. **Real-Time Balance Tracking**
   - Multi-account support
   - Live balance updates
   - Account-level details

3. **P&L Reports**
   - Monthly and yearly views
   - Income vs. expense breakdown
   - Visual charts

4. **Property/Person Expense Allocation**
   - Tag transactions by property
   - Percentage-based allocation
   - Detailed reporting

5. **Manual Entry Wizard**
   - 3-step guided flow
   - Smart defaults
   - Validation

6. **Transfer Between Accounts**
   - Simple two-account selection
   - Amount validation
   - Instant sync
