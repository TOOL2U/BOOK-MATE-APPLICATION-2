## 📊 CONTRAST RATIOS

### Background to Card Contrast
```
#121212 (BACKGROUND)
    ↓  Δ 0.42% lightness
#1A1A1A (CARD_PRIMARY)      ← Perfect subtle contrast
    ↓  Δ 0.34% lightness
#1C1C1C (CARD_SECONDARY)    ← Slightly lighter for inputs
    ↓  Δ 0.34% lightness
#1E1E1E (CARD_ELEVATED)     ← More prominent
    ↓  Δ 1.00% lightness
#242424 (CARD_HIGHLIGHT)    ← Most prominent (non-accent)
```

### Text Contrast
```
#FFFFFF (TEXT_PRIMARY)      on #121212 → 21:1 (AAA)
#B3B3B3 (TEXT_SECONDARY)    on #121212 → 11:1 (AAA)
#777777 (TEXT_MUTED)        on #121212 → 4.5:1 (AA)
```

### Brand Yellow Contrast
```
#FFF02B (BRAND_YELLOW)      on #121212 → 16:1 (AAA)
#000000 (BRAND_BLACK)       on #FFF02B → 21:1 (AAA)
```

## 🎯 QUICK TIPS

1. **Darkest to Lightest:**
   - BACKGROUND (#121212) is darkest
   - CARD_HIGHLIGHT (#242424) is lightest (non-accent)
   - BRAND_YELLOW (#FFF02B) is the brightest accent

2. **Subtle Borders:**
   - Always use rgba(255,255,255,0.06) for borders
   - Never use solid gray (#4D4D4D, #2A2A2A, etc.)

3. **Text Hierarchy:**
   - Headings = #FFFFFF (brightest)
   - Labels = #B3B3B3 (medium)
   - Hints = #777777 (dimmest)

4. **Contrast Depth:**
   - Each card layer is only slightly lighter than the previous
   - Creates subtle, premium depth perception
   - Maintains dark theme while providing clear hierarchy
